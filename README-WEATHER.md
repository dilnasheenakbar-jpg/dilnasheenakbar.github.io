# Weather Dashboard

A real-time weather dashboard application that fetches current weather and 5-day forecasts using the OpenWeather API.

## Features

✨ **Current Weather Display**
- Real-time temperature and conditions
- City name and country
- Weather icon representation
- "Feels like" temperature

📊 **Detailed Weather Information**
- Humidity percentage
- Atmospheric pressure
- Wind speed
- Cloud coverage
- Visibility distance

🔮 **5-Day Weather Forecast**
- Daily weather predictions
- Temperature forecasts
- Weather conditions
- Visual weather icons

🔍 **Search Functionality**
- Search weather by city name
- Real-time error handling
- Loading indicators
- Enter key support

📱 **Responsive Design**
- Works on desktop and mobile devices
- Smooth animations and transitions
- Modern gradient background
- Card-based layout

## How to Use

1. Open `weather-dashboard.html` in your browser
2. Enter a city name in the search box
3. Click "Search" or press Enter
4. View current weather and 5-day forecast

## API Used

- **OpenWeather API** (https://openweathermap.org/api)
- Free tier: 5-day forecast, current weather
- Units: Metric (Celsius)
